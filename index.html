rabbitMQ

复习rabbitMQ

阿里云centos7服务器安装

1： 安装erlang环境(rabbitMQ使用erlang开发)

2: 安装rabbitmq rpm包

3: 启动rabbitMq服务

4: 启用维护插件

5: 登陆服务器RabbitMQ 可视化控制台(安全组端口号记得配置15672)
http://ip:15672/

6: 程序连接需要设置新用户的访问权限(本次使用可视化平台)

  • 点击admin
  • 点击virtual Hosts
  • 点击 “ / “ 进入 (可以自己创建)
  • Permissions > Set permission 选择创建的用户名称
  • 点击 Set permission 按钮
